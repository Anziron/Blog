Core概念

LangChain框架的核心概念主要包括Chain、Memory和Prompt，这些是构建LLM应用的基础构建块。

Chain（链）​

Chain是LangChain中最基本的抽象之一，它将多个组件（如LLM、Prompt模板、其他Chain等）串联起来，形成一个完整的处理流程。

from langchain.chains import LLMChain
from langchain.llms import OpenAI
from langchain.prompts import PromptTemplate

# 创建一个简单的Chain
prompt = PromptTemplate(
    input_variables=["product"],
    template="为{product}写一个简短的广告语。",
)
llm = OpenAI(temperature=0.7)
chain = LLMChain(llm=llm, prompt=prompt)

# 使用Chain
result = chain.run(product="一款智能手机")
print(result)

Memory（记忆）​

Memory允许Chain在多轮交互中保持状态，这对于构建对话应用至关重要。

from langchain.chains import ConversationChain
from langchain.memory import ConversationBufferMemory

# 创建带有记忆功能的对话Chain
memory = ConversationBufferMemory()
conversation = ConversationChain(
    llm=OpenAI(temperature=0.7),
    memory=memory,
    verbose=True
)

# 进行多轮对话
response1 = conversation.predict(input="你好！")
response2 = conversation.predict(input="我想了解机器学习。")

Prompt（提示）​

Prompt是与LLM交互的文本指令，LangChain提供了强大的工具来创建和管理提示模板。

from langchain.prompts import ChatPromptTemplate, HumanMessagePromptTemplate

# 创建一个聊天提示模板
human_template = "回答以下问题: {question}"
human_message_prompt = HumanMessagePromptTemplate.from_template(human_template)
chat_prompt = ChatPromptTemplate.from_messages([human_message_prompt])

# 格式化提示
formatted_prompt = chat_prompt.format_prompt(question="什么是机器学习？").to_messages()

组合使用​

这三个核心概念通常一起使用，构建强大的LLM应用：

from langchain.chains import ConversationChain
from langchain.memory import ConversationBufferMemory
from langchain.prompts import PromptTemplate

# 自定义提示模板
template = """
以下是人类和AI助手之间友好的对话。
AI助手乐于助人、有创意、聪明而且非常友好。

当前对话:
{history}
人类: {input}
AI: 
"""

prompt = PromptTemplate(input_variables=["history", "input"], template=template)
memory = ConversationBufferMemory(return_messages=True)

conversation = ConversationChain(
    llm=OpenAI(temperature=0.7),
    prompt=prompt,
    memory=memory,
    verbose=True
)
