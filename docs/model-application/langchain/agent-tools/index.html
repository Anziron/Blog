Agent 工具调用实践

Agent（智能代理）是LangChain中一个强大的概念，它允许语言模型决定使用哪些工具来完成任务，并能根据工具执行结果做出后续决策。这使得LLM可以与外部世界交互，执行复杂的多步骤任务。

Agent的基本原理​

Agent的工作流程通常包括以下步骤：

  1. 观察: 接收用户输入和环境状态
  2. 思考: 根据观察决定下一步行动
  3. 行动: 选择并调用适当的工具
  4. 反馈: 获取工具调用结果并更新状态
  5. 重复: 根据新状态继续决策过程，直到完成任务

使用LangChain构建Agent​

1. 定义工具​

from langchain.agents import Tool
from langchain.tools import BaseTool
from langchain.utilities import SerpAPIWrapper, WikipediaAPIWrapper

# 使用预定义工具
search = SerpAPIWrapper()
wikipedia = WikipediaAPIWrapper()

# 定义工具列表
tools = [
    Tool(
        name="搜索",
        func=search.run,
        description="当你需要搜索互联网上的信息时有用"
    ),
    Tool(
        name="维基百科",
        func=wikipedia.run,
        description="当你需要查询百科知识时有用"
    )
]

# 自定义工具
class CalculatorTool(BaseTool):
    name = "计算器"
    description = "用于执行数学计算"
  
    def _run(self, query: str) -> str:
        try:
            return str(eval(query))
        except Exception as e:
            return f"计算错误: {e}"
  
    def _arun(self, query: str) -> str:
        return self._run(query)

tools.append(CalculatorTool())

2. 创建Agent​

from langchain.agents import initialize_agent, AgentType
from langchain.llms import OpenAI

# 初始化语言模型
llm = OpenAI(temperature=0)

# 创建Agent
agent = initialize_agent(
    tools,
    llm,
    agent=AgentType.ZERO_SHOT_REACT_DESCRIPTION,
    verbose=True
)

3. 运行Agent​

# 执行任务
result = agent.run("北京的人口是多少？同时，计算一下2023减去1949等于几？")
print(result)

Agent的高级用法​

1. 使用记忆功能​

from langchain.memory import ConversationBufferMemory

memory = ConversationBufferMemory(memory_key="chat_history")

agent_with_memory = initialize_agent(
    tools,
    llm,
    agent=AgentType.CONVERSATIONAL_REACT_DESCRIPTION,
    memory=memory,
    verbose=True
)

agent_with_memory.run("你好，我叫张三")
agent_with_memory.run("我的名字是什么？")

2. 自定义Agent​

from langchain.agents import Tool, AgentExecutor, ZeroShotAgent
from langchain.chains import LLMChain

# 定义提示模板
prefix = """你是一个智能助手，可以使用以下工具来回答问题:"""
suffix = """开始！

问题: {input}
{agent_scratchpad}"""

# 创建工具描述
tool_descriptions = "\n".join([f"{tool.name}: {tool.description}" for tool in tools])
prompt = ZeroShotAgent.create_prompt(
    tools, 
    prefix=prefix, 
    suffix=suffix,
    input_variables=["input", "agent_scratchpad"]
)

# 创建LLM链
llm_chain = LLMChain(llm=llm, prompt=prompt)

# 创建Agent
custom_agent = ZeroShotAgent(llm_chain=llm_chain, tools=tools)
agent_executor = AgentExecutor.from_agent_and_tools(
    agent=custom_agent,
    tools=tools,
    verbose=True
)

常见应用场景​

  • 信息检索和整合: 从多个来源获取信息并合成答案
  • 自动化工作流: 执行多步骤的业务流程
  • 代码助手: 编写、调试和执行代码
  • 个人助手: 日程安排、提醒、预订等

Agent技术是LLM应用中最具前景的方向之一，通过连接外部工具和系统，大大扩展了语言模型的能力边界。
