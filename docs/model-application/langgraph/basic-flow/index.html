基础流程图构建

LangGraph允许开发者以图的形式定义复杂的工作流程，使多步骤、多分支的AI应用构建变得更加直观。

基本概念​

在LangGraph中，工作流由以下几部分组成：

  • 节点(Nodes): 执行特定功能的组件
  • 边(Edges): 定义节点之间的连接和转换条件
  • 状态(State): 在节点之间传递的数据
  • 通道(Channels): 定义状态中可访问的数据子集

创建简单工作流​

1. 创建节点​

from langgraph.graph import StateGraph
from typing import TypedDict, Annotated, List
from langchain.schema import HumanMessage

# 定义状态类型
class ChatState(TypedDict):
    messages: List[dict]
    next: str

# 创建图实例
workflow = StateGraph(ChatState)

# 定义节点函数
def echo_node(state: ChatState) -> ChatState:
    """简单的消息回显节点"""
    messages = state["messages"]
    response = f"收到消息: {messages[-1]['content']}"
    return {"messages": messages + [{"role": "assistant", "content": response}]}

def router(state: ChatState) -> str:
    """路由节点，决定下一步去向"""
    message = state["messages"][-1]["content"].lower()
    if "结束" in message:
        return "end"
    else:
        return "echo"

# 添加节点
workflow.add_node("echo", echo_node)
workflow.add_node("router", router)

2. 定义边和条件​

# 添加边和条件
workflow.set_entry_point("echo")
workflow.add_edge("echo", "router")
workflow.add_conditional_edges(
    "router",
    # 根据router返回的字符串决定下一步
    {
        "echo": "echo",
        "end": "end"
    }
)
workflow.add_terminal_node("end")

3. 编译和执行图​

# 编译图
chain = workflow.compile()

# 执行工作流
response = chain.invoke(
    {
        "messages": [{"role": "user", "content": "你好，这是一条测试消息"}]
    }
)

print("工作流执行结果:")
for message in response["messages"]:
    print(f"{message['role']}: {message['content']}")

创建复杂工作流​

# 扩展节点功能
def analyze_node(state: ChatState) -> ChatState:
    """分析用户意图的节点"""
    messages = state["messages"]
    last_message = messages[-1]["content"]
  
    analysis = "这是一个普通查询"
    if "?" in last_message or "什么" in last_message:
        analysis = "这是一个问题"
    elif "帮我" in last_message or "请" in last_message:
        analysis = "这是一个请求"
      
    return {"messages": messages, "analysis": analysis}

def response_node(state: ChatState) -> ChatState:
    """根据分析生成回复"""
    messages = state["messages"]
    analysis = state.get("analysis", "未知类型")
  
    if analysis == "这是一个问题":
        response = "我会尽力回答您的问题。"
    elif analysis == "这是一个请求":
        response = "我会尽力满足您的请求。"
    else:
        response = "我理解了您的意思。"
      
    return {"messages": messages + [{"role": "assistant", "content": response}]}

# 创建更复杂的工作流
complex_workflow = StateGraph(ChatState)

complex_workflow.add_node("analyze", analyze_node)
complex_workflow.add_node("respond", response_node)
complex_workflow.add_node("router", router)

complex_workflow.set_entry_point("analyze")
complex_workflow.add_edge("analyze", "respond")
complex_workflow.add_edge("respond", "router")
complex_workflow.add_conditional_edges(
    "router",
    {
        "echo": "analyze",  # 返回到分析节点
        "end": "end"
    }
)
complex_workflow.add_terminal_node("end")

# 编译和执行
complex_chain = complex_workflow.compile()

工作流可视化​

LangGraph支持将工作流导出为可视化格式，便于调试和分享：

# 导出为Mermaid图表
from langgraph.viz import visualize

mermaid = visualize(workflow)
print(mermaid)
